# 互動意圖規範 - 企劃

本文件將遊戲企劃意圖轉化為互動意圖規範，方便企劃驗收與跨部門溝通。

## 文件用途

- 整理玩家操作、動作意圖、節奏、特效與音效

## 遊戲核心理念

- **目標**：玩家操控角色收集道具並避開敵人，累積分數，挑戰排行榜。
- **核心行為**：
  1. 查看簡單規則 → 理解操作
  2. 開始遊戲 → 角色互動與分數累計
  3. 分數即時更新 → 提供即時回饋
  4. 暫停 / 靜音 / 重玩 → 控制遊戲節奏
  5. 排行榜 → 提升重玩動機

## 角色攻擊（Warrior Attack）

### 動作目的

玩家操作角色攻擊敵人，提供力量感與反饋。

### 觸發方式

- 玩家點擊「攻擊」按鈕
- AI 自動近距離攻擊

### 前置條件

- 角色狀態為 idle
- 攻擊冷卻完成

### 動作意圖（Visual Intent）

- 揮劍展現力量感
- 命中瞬間呈現衝擊特效與音效
- 動作結束回 idle，自然流暢

### 動作長度

- 總時長：0.8 秒
- 階段：
  - 0.0–0.45 秒：揮劍前衝
  - 0.45–0.52 秒：命中特效生成、音效播放
  - 0.52–0.8 秒：角色回 idle

### Timeline

| 時間（秒） | 事件名稱 | 敘述             | 說明           |
| ---------- | -------- | ---------------- | -------------- |
| 0.45       | 音效播放 | 播放揮劍音效     | 突顯攻擊力量   |
| 0.50       | 特效生成 | 命中特效出現     | 命中瞬間衝擊感 |
| 0.52       | 傷害反饋 | 玩家看到傷害數值 | 動作高潮點     |

### 玩家互動流程

玩家點擊攻擊 → 揮劍動作 → 音效/特效同步 → 命中反饋 → 回 idle

## 技能釋放（Fireball Skill）

### 動作目的

玩家或 AI 角色釋放火球技能，提供視覺焦點與攻擊反饋。

### 觸發方式

- 玩家點擊技能按鈕
- AI 自動施放技能

### 前置條件

- 角色狀態為 idle
- 技能冷卻完成
- 角色血量 ≥ 1

### 動作意圖（Visual Intent）

- 火球從角色手中飛出，表現力量感與焦點
- 命中敵人時產生爆炸特效
- 動作結束回 idle，技能冷卻開始計時

### 動作長度

- 總時長：1.2 秒
- 階段：
  - 0.0–0.4 秒：角色施法準備
  - 0.4–0.9 秒：火球飛行
  - 0.9–1.2 秒：命中爆炸、回 idle

### Timeline

| 時間（秒） | 事件名稱 | 敘述         | 說明       |
| ---------- | -------- | ------------ | ---------- |
| 0.35       | 音效播放 | 施法音效     | 強化施法感 |
| 0.9        | 特效生成 | 火球爆炸特效 | 命中敵人   |
| 0.95       | 傷害反饋 | 顯示傷害數值 | 動作高潮   |

### 玩家互動流程

玩家點擊技能 → 施法動畫 → 火球飛行 → 命中特效 + 音效 → 傷害顯示 → 回 idle

## 角色升級彈窗（Warrior LevelUp Modal）

### 動作目的

告知玩家角色升級成功，並營造成就感。

### 觸發方式

- 系統事件：onLevelUp(newLevel)

### 前置條件

- 無其他高優先 modal 開啟

### 動作意圖（Visual Intent）

- 彈出特效 + 光暈 + 文字浮現
- idle 動畫停留時間由系統控制，玩家可點擊確認

### 動作長度

- enter：0.45 秒
- idle：1.8 秒
- exit：0.4 秒

### Timeline

| 時間（秒） | 事件名稱 | 敘述           | 說明       |
| ---------- | -------- | -------------- | ---------- |
| 0.20       | 音效播放 | 播放升級音效   | 增強成就感 |
| 0.28       | 特效生成 | 光效或星爆特效 | 視覺焦點   |

### 玩家互動流程

升級事件觸發 → enter 動畫 → 音效/光效 → idle 停留 → 玩家確認 → exit 動畫結束

## 角色防禦動作（Warrior Block）

### 動作目的

提供防禦操作回饋，增加策略感。

### 觸發方式

- 玩家點擊「防禦」按鈕
- AI 或系統判定自動防禦

### 前置條件

- 角色狀態為 idle

### 動作意圖（Visual Intent）

- 盾牌舉起提供安全感
- 成功防禦時有衝擊回饋
- 動作結束回 idle 或維持防禦姿態

### 動作長度

- 總時長：0.8 秒
- 階段：
  - 0.0–0.3 秒：舉盾準備
  - 0.3–0.5 秒：防禦命中反饋
  - 0.5–0.8 秒：回 idle 或維持防禦

### Timeline

| 時間（秒） | 事件名稱 | 敘述         | 說明       |
| ---------- | -------- | ------------ | ---------- |
| 0.3        | 特效     | 命中防禦光效 | 提供回饋   |
| 0.35       | 音效     | 防禦成功提示 | 強化安全感 |

### 玩家互動流程

玩家點擊防禦 → 舉盾準備 → 防禦命中 → 特效/音效回饋 → 回 idle

## 技能冷卻提示（Skill Cooldown UI）

### 動作目的

提醒玩家技能冷卻完成，提高操作直覺。

### 觸發方式

- 系統判斷技能冷卻結束

### 前置條件

- 無其他 UI 干擾

### 動作意圖（Visual Intent）

- 按鈕亮起並閃爍，清楚提示可使用
- 動作輕快，強化玩家操作感受

### 動作長度

- 總時長：0.5 秒
- 階段：
  - 0.0–0.2 秒：按鈕漸亮
  - 0.2–0.5 秒：閃爍提示完成

### Timeline

| 時間（秒） | 事件名稱 | 敘述         | 說明         |
| ---------- | -------- | ------------ | ------------ |
| 0.2        | 音效提示 | 技能可用音效 | 增強操作直覺 |

### 玩家互動流程

技能冷卻完成 → 按鈕亮起 → 閃爍提示 → 玩家使用技能

## 任務完成彈窗（Quest Complete Modal）

### 動作目的

提示玩家任務完成，展示獎勵。

### 觸發方式

- 系統事件：任務達成

### 前置條件

- 無其他高優先 modal 開啟

### 動作意圖（Visual Intent）

- 彈窗升起，展示任務內容與獎勵
- 動作流暢，成就感明確

### 動作長度

- enter：0.4 秒
- idle：1.6 秒
- exit：0.3 秒

### Timeline

| 時間（秒） | 事件名稱 | 敘述           | 說明         |
| ---------- | -------- | -------------- | ------------ |
| 0.2        | 光效生成 | 彈窗光效或星爆 | 提供視覺焦點 |
| 0.25       | 音效播放 | 成就提示音效   | 增強成就感   |

### 玩家互動流程

任務完成 → enter 動畫 → 光效/音效 → idle 顯示內容 → 玩家確認 → exit 動畫
